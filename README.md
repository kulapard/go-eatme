## Install ##
### OS X / Linux ###
Using [Homebrew](http://brew.sh/)
```
brew install kulapard/tap/eatme
```

## Update ##
### OS X / Linux ###
Using [Homebrew](http://brew.sh/)
```
brew upgrade kulapard/tap/eatme
```

## Run ##
```
eatme COMMAND
```
