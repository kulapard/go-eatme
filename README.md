## Install ##
### OS X / Linux ###
Using [Homebrew](http://brew.sh/) (OS X) or [Linuxbrew](http://linuxbrew.sh/) (Linux)
```
brew install kulapard/tap/go-eatme
```

## Update ##
### OS X / Linux ###
Using [Homebrew](http://brew.sh/) (OS X) or [Linuxbrew](http://linuxbrew.sh/) (Linux)
```
brew upgrade kulapard/tap/go-eatme
```

## Run ##
```
eatme COMMAND
```
